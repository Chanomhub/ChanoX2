## Reporting a Vulnerability

We take the security of ChanoX2 seriously. If you have discovered a security vulnerability, we appreciate your help in disclosing it to us in a responsible manner.

**Do not file a public issue.**

Please report any security vulnerabilities by emailing **contact@chanomhub.com**.

We will attempt to acknowledge your report within 48 hours, and proper steps will be taken to address the issue.
